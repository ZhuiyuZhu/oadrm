# results/figures/ — 论文图表

## 中文

本目录存放 OADRM 论文中的**可视化图表**（高分辨率 PNG / PDF）。

| 文件名 | 对应论文图 | 内容说明 |
|-------|-----------|---------|
| `organ_drug_heatmap.png` | Figure 3a | 8 个器官 × Top 候选化合物的 mCS 评分热图 |
| `homo_vs_hetero_scatter.png` | Figure 3b | 同源 vs. 异源 mCS 散点图（高特异性候选聚集于左上象限） |

### 生成方式
所有图表均由 `scripts/` 中的分析脚本自动生成（基于 `matplotlib` + `seaborn`），源数据位于 `results/tables/`。

### ⚠️ 说明
- 若需修改图表样式，请直接运行对应脚本重新生成，不建议手动编辑图片。
- 投稿级矢量图（PDF/SVG）可在脚本中通过 `plt.savefig('xxx.pdf')` 输出。

---

## English

This directory stores **publication-quality figures** (high-resolution PNG / PDF) for the OADRM manuscript.

| Filename | Manuscript Figure | Description |
|---------|------------------|-------------|
| `organ_drug_heatmap.png` | Figure 3a | Heatmap of mCS scores across 8 organs × top candidate compounds |
| `homo_vs_hetero_scatter.png` | Figure 3b | Homologous vs. heterologous mCS scatter plot (high-specificity candidates cluster in the upper-left quadrant) |

### Generation
All figures are auto-generated by analysis scripts in `scripts/` (using `matplotlib` + `seaborn`), with source data in `results/tables/`.

### ⚠️ Notes
- To modify figure styles, rerun the corresponding scripts rather than editing images manually.
- Vector graphics (PDF/SVG) for submission can be exported via `plt.savefig('xxx.pdf')` in the scripts.
