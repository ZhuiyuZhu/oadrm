# paper/ — 论文手稿与补充材料

## 中文

本目录存放与 OADRM 论文撰写相关的**文档、框架和版本历史**。

| 文件 | 说明 |
|------|------|
| `manuscript_framework.md` | 论文框架与章节大纲（含 Figure/Table 占位符） |
| `manuscript_v1.docx` | 完整手稿（Word 格式，含图表插槽） |
| `supplementary_tables.xlsx` | 补充表格：GTEx 验证详情、完整候选药物列表、LINCS 细胞系映射表 |
| `response_to_reviewers.md` | 审稿回复模板（投稿后维护） |

### 使用说明
- `manuscript_framework.md` 为**半成品/工作版本**，包含待填充的统计值和空白图注（如 "【此处留白：待统计检验后填入】"）。
- 正式投稿版本将从本目录迁移至期刊投稿系统，GitHub 保留的是**可审计的开发历史**。
- 补充材料中的大表格（如完整 139 候选药物信息）建议同步上传至 **Zenodo/Figshare** 并在此引用 DOI。

## English

This directory stores **manuscript drafts, outlines, and supplementary materials** for the OADRM publication.

| File | Description |
|------|-------------|
| `manuscript_framework.md` | Manuscript outline and section framework (with Figure/Table placeholders) |
| `manuscript_v1.docx` | Full manuscript draft (Word format, with figure slots) |
| `supplementary_tables.xlsx` | Supplementary tables: GTEx validation details, full candidate list, LINCS cell-line mapping |
| `response_to_reviewers.md` | Response-to-reviewers template (maintained post-submission) |

### Usage Notes
- `manuscript_framework.md` is a **working draft** containing placeholders for pending statistics and blank figure captions (e.g., "【此处留白：待统计检验后填入】").
- The final submission version will be transferred to the journal system; GitHub retains the **auditable development history**.
- Large supplementary tables (e.g., full 139-candidate drug information) should be uploaded to **Zenodo/Figshare** and referenced by DOI here.
